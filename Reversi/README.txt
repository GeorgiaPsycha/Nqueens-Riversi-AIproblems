Reversi-Minimax

Για την υλοποίηση του προβλήματος 3, Reversi (Othello) με Minimax αλγόριθμο, καθώς και για το debugging του χρησιμοποιήσαμε το ide Intellij IDEA.
Για να τρέξετε εσείς το πρόγραμμα αρκεί η αντιγραφή των 7 κλάσεων από τον φάκελο src και η επικόληση στον αντίστοιχο φάκελο σε δικό σας project σε οποιοδήποτε ide αλλίως με την εντολή open (ή import ανάλογα με το ide που θα χρησιμοποιήσετε).
